## AIエージェントへ渡すプロンプト

### デモ1：PC向けカードレイアウト

> Chrome DevToolsを使って、現在開いているイベント申し込みページを確認してください。3つの参加プランを横並びのカードとして見やすく整え、カードの高さとボタンの位置を揃えてください。既存ページの配色や角丸のデザインは維持してください。修正後はページを再読み込みし、スクリーンショットを撮って表示を確認してください。必要であれば、確認結果をもとに追加で調整してください。

### デモ2：スマートフォン対応

> Chrome DevToolsを使って、現在のページを幅375pxのスマートフォン表示で確認してください。横スクロールや要素の重なりが発生しないように修正し、参加プランのカードは縦一列、フォームの入力欄と送信ボタンは画面幅に合わせて表示してください。修正後は幅375pxと1280pxの両方で表示を確認し、PC表示に影響がないことも確認してください。

### デモ3：フォームの実装

> 申し込みフォームに入力チェックを実装してください。名前とメールアドレスを必須とし、未入力で送信した場合は各入力欄の近くにエラーメッセージを表示してください。正しく入力された場合はフォームを非表示にし、申し込み完了メッセージを表示してください。実際のサーバー通信やページ遷移は必要ありません。実装後は、未入力の場合と正しく入力した場合の両方をChrome上で操作し、意図した表示になることを確認してください。
